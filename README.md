# 🦁 Schnuppertag – Vibe Coding

Heute baust du mit KI deine eigene Website – ganz ohne Vorkenntnisse!

---

## 🛠️ Tool: Perplexity

👉 **[chat.mistral.ai](https://chat.mistral.ai)** öffnen – kostenlos, kein Login nötig

---

## 📋 Schritt für Schritt

**1. Perplexity öffnen**
Gehe auf **[chat.mistral.ai](https://chat.mistral.ai)** – einfach losschreiben, kein Account nötig!

**2. Den Code holen**
Kopiere diesen Text und füge ihn in den Chat ein:

> „Ich habe eine Website mit einer Datei index.html und style.css. Es gibt ein Bild namens lion.png. Mach die Website schöner – füge einen coolen Titel hinzu, einen farbigen Hintergrund und einen lustigen Text über den Löwen. Gib mir nur den fertigen HTML Code."

**3. Den Code kopieren**
Perplexity antwortet mit Code. Klicke auf **„Copy"** beim Codeblock.

**4. In VS Code einfügen**
Öffne die Datei `index.html`, markiere alles (`Ctrl+A`) und füge den neuen Code ein (`Ctrl+V`).

**5. Anschauen**
Speichere (`Ctrl+S`) und öffne die Datei im Browser – deine neue Website!
